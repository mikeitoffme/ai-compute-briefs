# AI compute briefs

Этот репозиторий содержит код и данные для двух аналитических ответов руководителю. 
Оценка приведена по состоянию на **май 2026**, сбор данных сделан в июне 2026.

- **Бриф 1 — Парк GPU.** Сколько вычислительных ускорителей у OpenAI и Anthropic и как их парк делится между обучением моделей и инференсом.
- **Бриф 2 — Экономика инференса.** Зарабатывает ли Atlas Cloud, продавая DeepSeek V4 Pro по $1,68 / 1M входных и $3,38 / 1M выходных токенов.

## Короткие ответы

- **Бриф 1.** В эксплуатации сегодня порядка **2,0 млн H100-эквивалентов у OpenAI и 1,7 млн у Anthropic** (диапазон оценки 1,0–3,0 млн). Это H100-эквивалент производительности, а не число физических чипов — у Anthropic парк в основном на ASIC (Trainium, TPU). OpenAI делит парк между типами задач примерно поровну (≈48% инференс), Anthropic больше отдает для инференса (≈65%). Анонсы на десятки ГВт мощности в новостях — это про контракты на 2026–2029, а не имеющиеся сейчас в распоряжении мощности.
- **Бриф 2.** **Компания зарабатывает со значительным запасом по марже:** валовая маржа на вычислениях ≈ **80–90%** (входящие токены ~93%, исходящие токены ~82%). Себестоимость инференса ≈ $0,12/M вход и $0,61/M выход против цены $1,68 / $3,38. Маржа положительна.

## Как воспроизвести цифры

**Все числа (без зависимостей):**
```bash
python3 model.py
```
Выводит данные по парку OpenAI/Anthropic, сплит между обучением/инференсом, себестоимость инференса, маржу по сценариям и независимую bottom-up TCO-проверку. Эти же числа вынесены на слайды.

**Notebook (с пояснениями по шагам):**
```bash
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
jupyter notebook notebook.ipynb
```

## Методология (откуда взялись цифры)

- **Бриф 1.** Парки GPU нельзя сравнивать «по штукам» между указанными компаниями: у Anthropic в основном специализированные ускорители (AWS Trainium, Google TPU), а не GPU. Каждый ускоритель приводится к **H100-эквиваленту** по производительности и суммируется. Откуда брались допущения: независимая оценка Epoch AI (~1,1 млн H100-экв у OpenAI, сер. 2025), публикации Anthropic/AWS/Google, аналитика SemiAnalysis.
- **Бриф 2.** Себестоимость токена = стоимость gpu-ноды в час ÷ (пропускная способность × загрузка), отдельно для prefill (вход) и decode (выход). Три сценария (эффективный / базовый / консервативный) стресс-теста присутствуют. Стоимость арендной gpu ($3/GPU-ч) перепроверена доп. расчетом bottom-up TCO (≈$19/ч за ноду в собственности).
- Все исходные значения, диапазоны и ссылки находятся в [`sources.md`](sources.md).

## Оговорки

- Состав парков ускорителей OpenAI и Anthropic не раскрывают, поэтому это **оценки на базе открытых данных**, и вместо «точных/конкретных» цифр стоят диапазоны low/base/high.
- H100-эквиваленты это приближённый пересчёт по пиковой производительности; для инференса узкое место — это память, а не пиковые FLOPs.
- Маржа Atlas указана **валовая, по стоимости вычислений** (на каждый дополнительный запрос к модели). Это базовый сценарий (~80–90%); в пессимистичном сценарии общая маржа остаётся положительной, но на выходных токенах сжимается почти до нуля (decode — узкое место). Операционная прибыль будет ниже: в слайдах и расчетах не учтены простой/недозагрузка, бесплатные тарифы, сетевой трафик, поддержка, R&D и т.п.
- Цена у Atlas ($1,68/$3,38) согласуется со значениями у конкурентов премиум сегмента ($1,74/$3,48, на ~3% выше).
